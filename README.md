# 🔐 Cybersecurity Awareness Chatbot (POE Part 3)

## 📌 Project Overview
This project is a Cybersecurity Awareness Chatbot developed using **C# Windows Forms**.  
It is designed to educate users about online safety while providing interactive features such as a quiz game, task manager, and activity logging system.

---

## 🚀 Features

### 💬 1. Chatbot with NLP Simulation
- Uses keyword detection to simulate Natural Language Processing (NLP)
- Recognises intents like:
  - Password safety 
  - Phishing
  - Scams
  - Privacy
  - Tasks
  - Quiz requests

---

### 🧠 2. Cybersecurity Quiz Game
- Interactive quiz with multiple-choice and true/false questions
- Tracks user score
- Provides instant feedback after each answer
- Encourages cybersecurity learning

---

### 🗂️ 3. Task Manager (SQL Database Integration)
- Users can add cybersecurity-related tasks
- Tasks are stored in a **SQL Server database**
- Tasks can be viewed and deleted
- Displayed using a **DataGridView**

---

### 📋 4. Activity Log System
- Tracks all important user actions:
  - Messages sent
  - Tasks added
  - Quiz activity
  - System responses
- Displays last 5–10 actions for clarity

---

### 🎨 5. Graphical User Interface (GUI)
- Built using Windows Forms (C#)
- Includes:
  - Chat window
  - Input textbox
  - Buttons for actions
  - DataGridView for tasks
  - Activity log viewer

---

## 🛠️ Technologies Used
- C#
- Windows Forms (WinForms)
- SQL Server (Database)
- ADO.NET (Database connection)
- Visual Studio 2022

---

## 🧩 How It Works
1. User types a message into the chatbot
2. NLP simulation detects intent using keywords
3. Chatbot responds with cybersecurity advice or actions
4. Tasks are stored in SQL database
5. Quiz system tracks knowledge progress
6. Activity log records all actions

---

## 📊 Database Structure

**Table: Tasks**
- Title (nvarchar)
- Description (nvarchar)
